How to Apply Links in a website?

We can apply links to an html page by anchor tag the location where we want user to go is written inside href attribute. After that inside we can put any content (images, heading, paragraphs, etc.. are all allowed) if user click on that he/she will reach the location you specified.

For Example

<a href="www.google.com">This is a Link</a>


we can also give link of a html page by giving file name in href attribute as:- 

<a href="index.html">This is a Link of HTML Page</a>


Target attribute

It specifies where the specified location will open.
Properties of target attribute -

  • _self  Opens the specified location in itself
  • _blank  Opens the specified location in a new tab
  • _parent Opens the specified location in parent tab
  • _top  Opens the specified location in the top tab

Note = _parent and _top properties are used in iframes we will discuss them in iframe tutorial.
