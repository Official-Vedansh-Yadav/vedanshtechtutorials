HTML TAG

HTML tag is the root tag of the html file this is parent tag of head and body tags. A html document always start with opening HTML tag (<html>) and ends with closing HTML tag (</html>).



HEAD TAG

HEAD Tag is like a head of the web site it contains all Meta tags, link tags and title tag. So it is parent tag of all of these tags. It is also very important tag it is necessary to write as html tag. It closes before BODY tag.



TITLE TAG

Title tag is used to give a title to the page we write title we want to give inside title tag. Like :-- 

 <title>Vedansh Tech Tutorials</title>

BODY TAG

BODY tag contains body of a web page that is rendered by the browser. You can see image :-

Anything we will write inside body tag is rendered on the website body. Let's see praticle

Code to be write :- 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">

Important Topics

  • head & body tags are children tags of html tag.
  • HTML tag is the parent of head & body tag.
  • Most of the html elements have opening and closing tag with content in between opening and closing tags.
  • Some HTML tags have no content. These are called Empty elements. e.x. -> <br> tag.
  • Basic syntax for html element = opening tag + content + closing tag
    Opening tag = <tag_name>
    Closing tag = </tag_name>
    for ex. = <body>...</body>


What are Parent Tags and Child tags?

A Tag containing any tag inside it is called parent tag of that tag which it containing and the tag is contained by any other tag is called child tag of that tag for example html tag is parent tag head and body tags.
