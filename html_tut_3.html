Comments

Comments are used to mark text which should not be rendered or shown on browser. They can help in identifying tags
Syntax =

For Example :- 

<!-- This is a Comment -->

It will be not rendered on the page but shown in the page source



Heading tags

Heading tag is used to mark heading in HTML. From h1 to h6, we have these tag to make heading and the size of text decreases from h1 to h6

  • <h1>Biggest heading</h1>
  • <h2>might smaller heading</h2>
  • <h3>more smaller heading</h3>
  • <h4>smaller heading than h3</h4>
  • <h5>smaller heading than h4</h5>
  • <h6>smallest heading</h6>

For Example

<h1>Vedansh Tech Tutorials</h1>
<h2>Vedansh tech Tutorials</h2>
<h3>Vedansh Tech Tutorials</h3>
<h4>Vedansh Tech Tutorials</h4>
<h5>Vedansh Tech Tutorials</h5>
<h6>Vedansh Tech Tutorials</h6>

OutPut :-



Paragraph Tag

Paragraph tag is used to add paragraphs to an html page. 

For ex. = <p>This is a paragraph</p>



br Tag

The br tag is used to create line breaks means for breaking lines in an html document. It is an empty element.

Syntax = <br>



hr Tag

The hr tag is used to create a horizontal line often used to separate the content. It is also a empty tag like br tag. 

Syntax = <hr>
